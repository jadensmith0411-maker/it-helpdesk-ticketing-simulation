# IT Help Desk Ticketing Simulation

This project simulates a real-world IT help desk environment using Spiceworks, where I created and resolved technical support tickets across networking, hardware, software, and account access issues.

# Skills Demonstrated
- IT troubleshooting and diagnostics
- Ticket lifecycle management
- User communication
- Issue escalation (Tier 1 → Tier 2)
- Technical documentation

# Tools Used
- Spiceworks
- Windows OS

# Example Tickets
- WiFi connectivity issues
- VPN connection problems
- Printer troubleshooting
- Software not opening
- Monitor/display issues

# Project Purpose
To practice real-world IT support workflows and build hands-on experience in a help desk environment.
